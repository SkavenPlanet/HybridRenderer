# HybridRenderer

Hybrid Deferred Rendering Pipeline using Unity's SRP feature.

Features:
- Realtime Raytraced Mirror Specular & Diffuse GI (DDGI)
- Raytraced Shadows
- Raytraced AO
- Voxel Light Culling
- Sky Rendering (Precomputed Atmospheric Scattering)
- Unified Lighting Path
